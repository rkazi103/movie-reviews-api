## Run Locally

Clone the project

```bash
  git clone https://github.com/rkazi103/movie-reviews-api/
```

Go to the project directory

```bash
  cd movie-reviews-api
```

Install dependencies

```bash
  pipenv install
```

Start the server

```bash
  python manage.py runserver
```
